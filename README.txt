
Estrutura:

    * server - vindo do bazaar oficial do OpenERP 6.1
    * web - vindo do bazaar oficial do OpenERP 6.1 (com modificações visuais e funcionais da Integra)
    * addons - vindo do bazaar oficial do OpenERP 6.1 (com modificações funcionais da Integra)
    * integra - módulos desenvolvidos pela Integra
    * integra_extra - módulos de terceiros, utilizados pela Integra

    conf_openerp.conf.template - template do arquivo de configuração padronizado
    openerp-server - link para o openerp-server na pasta server
